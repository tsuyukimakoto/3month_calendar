# 3-Month Calendar Widget (threemonthcal)

A simple macOS widget that displays a three-month calendar on your desktop.

## How to Install

1. **Download:** Download the latest `.zip` file from the [Releases] page.
2. **Move to Applications:** Unzip the file and move the app to your `/Applications` folder.
3. **Initial Launch:** Open the app once to register the widget with macOS.

## How to Add the Widget

1. Right-click your desktop and select **"Edit Widgets..."**
2. Search for **"threemonthcal"** in the widget gallery.
3. Choose either the **Medium** or **Large** size and drag it to your desktop or Notification Center.

## Configuration

You can customize the calendar by right-clicking the widget and selecting **"Edit 'threemonthcal'"**:

* **Start of Week:** Toggle between **Monday** and **Sunday** starts.

2. Set:
- `TEAM_ID`
